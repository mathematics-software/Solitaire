Pyramid Solitaire - Without a Mouse!

Key Commands


  • Left Arrow: Move to the card to the left of the current card, or to the deck from the discard pile.
  • Right Arrow: Move to the card to the right of the current card, or to the discard pile from the deck.
  • Up Arrow: Move to the first visible card on the row of the pyramid directly above the current card. This button has no effect if you are on the deck or discard piles.
  • Down Arrow: Move to the first visible card on the row of the pyramid directly below the current card. This button has no effect if you are on the deck or discard piles.
  • Enter: If you not on the deck, the current card is selected. Once two matching cards (or one king) are selected, they are removed. If you are on the deck, the uppermost card is drawn if possible.
  • D: If you are on one of the rows of the pyramid, this moves you to the deck. If you are on the deck, it moves you to the last selected card on the pyramid. This card may have been removed, so the arrow keys may have to be used to obtain a visible card.
  • Ctrl+N: Starts a new game. There is no prompt to cancel this action, so be sure you are completely finished before performing this action!
